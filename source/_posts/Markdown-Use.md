---

[**下方速查表原址**](https://markdown.com.cn/cheat-sheet.html)
| 元素 | Markdown 语法 |
| ----------- | ----------- |
| 标题 | # ## ### #### ##### ###### <br> *PS：在文本下方添加任意数量的 `==` 号可以标识`一级`标题，添加 `--` 号可以标识`二级`标题* |
| 换行 | 两个`空格` 或 \<br> |
| 段落 | 空白行 | 
| 粗体 | ** &nbsp;&nbsp; ** |
| 斜体 | * &nbsp;&nbsp; * |
|代码块 | \`\`\` &nbsp;&nbsp; \`\`\` <br> 或 内容整体缩进一个 `Tab` |
| 删除线 | ~~ &nbsp;&nbsp; ~~ |
| 分割线 | --- &nbsp;&nbsp;或&nbsp;&nbsp; *** &nbsp;&nbsp;或&nbsp;&nbsp; ___ |
| 代码 或 禁用自动URL链接 | \` &nbsp;&nbsp; \` |
| 有序列表 | 1. &nbsp;&nbsp; 2. &nbsp;&nbsp; 3. |
| 无序列表 | 连续的&nbsp; - &nbsp;或&nbsp;&nbsp; * &nbsp;或&nbsp;&nbsp; + <br> *PS：如果要在`保留列表连续性`的同时在列表中`添加另一种元素`，请将该元素`缩进四个空格`或`一个制表符`* |
| 链接 | `[rubbishBin](https://physical2.github.io/rubbishBin/)` 或 \<<https://physical2.github.io/rubbishBin>\> |
| 引用块 | > <br> > <br> >> (嵌套块引用) |
| 图片 | `![alt text](/source/logo.png)` |
| 表格 | （就是你眼前这种） |
| 待办 | - [x]  或 - [ ] |
| Emoji ✨ | [Emojipedia](https://emojipedia.org/)
  
*PS：Markdown 可以内嵌 HTML 标签*
